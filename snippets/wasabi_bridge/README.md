To use **one_inventory** with **wasabi_bridge**, follow these steps:

1. Navigate to the following directory: `wasabi_bridge\inventories`

2. Place the **one_inventory** bridge file/folder inside the `inventories` directory.

3. Save the file and restart the resource.

**wasabi_bridge** will now use **one_inventory** as its inventory system.
