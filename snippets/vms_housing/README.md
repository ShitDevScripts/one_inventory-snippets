To use **one_inventory** with **vms_housing**, follow these steps:

1. Navigate to the following directory: `vms_housing\integration\[inventory]`

2. Place the **one_inventory** bridge file/folder inside the `inventory` directory.

3. Go to **config.integrations.lua** and add this to the DetectActiveInventory: **'one_inventory'**

4. Save the file and restart the resource.

**vms_housing** will now use **one_inventory** as its inventory system.
